# HOVER DODGE

> 壁と敵をよけ続けろ — ネオン・サイバーパンク調のフラッピーバード系回避ゲーム。

Three.js + WebGL で描画する一画面・タップで遊べるアーケードシューターです。ブラウザだけで動き、PWA としてホーム画面に追加すればオフラインでもプレイできます。フェーズに応じて重厚化する手続き合成 BGM 付き。

## 🎮 遊び方

- **タップ / SPACE** — 自機を上昇させる（重力で落ちる）
  - `FORWARD SHOT` 装備時はタップ時に前方弾も発射
- **PULSE / `Z`** — 範囲攻撃。クールダウン中の敵を一掃
- **`M`** — 音 (BGM + SE) ON / OFF（設定は保存）
- **`ESC`** — ゲームオーバー画面からタイトルへ
- スコア = 生存秒数 + 各種ボーナス（後述）

ゴールはひたすら長く生き残ること。死んだら即 `RETRY` ボタンでやり直し、自己ベストとXPはローカルに保存されます。

縦画面では `SHOP` ボタンが表示され、貯めた XP でスキルを購入・装備できます。

## 📈 フェーズ進行

スコアに応じて4段階に難易度が変化し、**BGM もそれに合わせて重厚化**します。

| Phase | スコア帯 | 敵構成 | BGM |
| :---: | :---: | --- | --- |
| **1** | 0 – 20 sec   | `WALLS` 壁のみ。練習フェーズ            | アンビエントパッドのみ |
| **2** | 20 – 50 sec  | `BULLETS` プラズマ弾型の敵が登場          | + ベース + キック |
| **3** | 50 – 100 sec | `SEEKERS` 追跡型シーカーが混入            | + リードリフ |
| **4** | 100 sec +    | `CHAOS` 全敵 + 最高速度                   | フルミックス |

フェーズ遷移時にはバナーと SE が再生され、BGM もスムーズにクロスフェードします。縦画面のタイトルでは自己ベストと**到達済みフェーズ**が可視化されます。

## 🎁 スコア・ボーナス

| アクション | ボーナス（スコア） |
| --- | :---: |
| 壁スレスレ通過（隙間中央から外側 30 % 以内） | **+10** |
| 極スレスレ通過（隙間中央から外側 15 % 以内） | **+25 NEAR!** |
| PULSE で `BULLET` 撃破 | **+15** |
| PULSE で `SEEKER` 撃破 | **+20** |
| 前方弾 (FORWARD SHOT) で撃破 | **+10〜15** |
| ノーパルス継続（5秒以上） | `PURIST` 表示 |

時間スコアと合算され、自己ベスト（秒）として記録されます。

## ✨ XP & ショップ

ランごとに XP が貯まり、縦画面の `SHOP` ボタンから**装備型スキル**を購入できます。

### XP 獲得レート

| ソース | XP |
| --- | :---: |
| 生存（時間経過） | +2 / 秒 |
| PULSE / 前方弾による撃破 | +5 |
| 壁スレスレ通過 | +3 |
| 極スレスレ通過 (NEAR!) | +8 |
| PHASE 4 初到達 | +60 |

典型的なランで 100 〜 400 XP 獲得。死亡時に `localStorage` に保存されます。

### スキル一覧

| スキル | 種別 | 価格 | 効果 |
| --- | --- | :---: | --- |
| **FORWARD SHOT** | 🟣 ATTACK  | 150 XP | タップで前方弾発射（CD 0.4 s）。敵に当たれば即撃破 |
| **RAPID PULSE**  | 🟣 ATTACK  | 250 XP | PULSE のクールダウンを 30 % 短縮（5 s → 3.5 s） |
| **SHIELD**       | 🟢 DEFENSE | 280 XP | ラン中の最初の 1 ヒットを吸収。周囲の敵も巻き込み撃破 |
| **MAGNET**       | 🔵 PASSIVE | 300 XP | NEAR-MISS 判定範囲を 50 % 拡大、ボーナス頻度 UP |
| **MEGA PULSE**   | 🟣 ATTACK  | 350 XP | PULSE の攻撃範囲を 30 % 拡大（130 → 170） |
| **SMALLER HITBOX** | 🔵 PASSIVE | 420 XP | 当たり判定 33 % 縮小（面積で約 55 % 減） |
| **XP BOOST**     | 🔵 PASSIVE | 500 XP | 獲得 XP 全体を 25 % 増加 |
| **PHOENIX**      | 🟢 DEFENSE | 600 XP | 死亡時に 1 度だけ復活、3 秒間無敵 |

**装備枠は 2 つ**。所持スキルのうち任意の 2 つを装備できます。組み合わせで戦略が変わります（例: `RAPID PULSE + MEGA PULSE` = 攻撃特化、`SHIELD + PHOENIX` = 実質 3 ライフ、`MAGNET + XP BOOST` = XP 稼ぎ）。

## 🎨 SKIN（自機カスタマイズ）

縦画面の **SKIN** ボタンから、宇宙船のカラースキンを 6 種類から選択できます。タップで即座に反映、選択は `localStorage` に保存され再起動後も維持されます。

| スキン | アクセントカラー |
| --- | --- |
| **CYAN**    | `#00e8ff` (デフォルト) |
| **MAGENTA** | `#ff2bd6` |
| **EMERALD** | `#44ff99` |
| **FIRE**    | `#ff7733` |
| **VOID**    | `#aa44ff` |
| **GOLD**    | `#ffdd44` |

各スキンは船体・翼・コクピット・エンジン・炎・shipLight の全色を統一的に塗り替えます（PULSE / 前方弾はスキル効果なのでスキンとは独立）。新スキンを追加するには `SHIP_SKINS` 配列に 1 エントリ追加するだけです。

### 新スキルを追加する

`SKILL_DEFS` に1エントリ追加し、効果コードを `hasSkill('id')` ガードで実装すれば終わりです。詳細は `index.html` のコメント参照。

## 🎵 BGM

外部音源ファイルを使わず、**WebAudio で手続き合成**しています。

- **4 声構成**: パッド（持続）/ ベース（鋸波）/ ドラム（ノイズ+サイン）/ リード（矩形波）
- **共通クロック**で常時走り、フェーズごとに `GainNode` を 1.6 秒線形ランプで上下 → 違和感のないクロスフェード
- **102 BPM、Am – F – C – G** の 4 小節ループ。16 ステップのリードリフは 2 ループに 1 回オクターブ上げてバリエーション
- **マスター → DynamicsCompressor → 出力** で SE 重畳時にもクリップなし
- タブ非アクティブからの復帰時は **past-due ノートをスキップ**して追いつく

START 押下で開始、死亡時にフェードアウト、RETRY で PHASE 1 から再生。

## 🛠 技術スタック

- **Three.js** (r160) — シーン / ジオメトリ / ライティング / ポストプロセス（UnrealBloomPass）
- **Vanilla JS (ES Modules)** — ゲームロジック、入力、状態管理
- **Canvas 2D** — HUD オーバーレイ
- **Web Audio API**:
  - SE（flap / pulse / hit / milestone / phase-up / bonus）— OscillatorNode + ノイズバッファで動的生成
  - BGM — 4 ボイスのレイヤードシンセ、lookahead スケジューラ
- **CSS Animations** — タイトル・ロゴ・縦画面オーバーレイ全般、グリッチ・スキャンライン等
- **Service Worker** — オフライン / PWA 対応（`hoverdodge-vX.Y.Z` キャッシュキーで自動更新）
- **localStorage**:
  - `hd_hi` — 自己ベスト（秒）
  - `hd_mute` — 音設定
  - `hd_xp` — 累積 XP
  - `hd_owned` — 所持スキル ID 配列
  - `hd_equip` — 装備スキル ID 配列（最大 2 枠）
  - `hd_skin` — 選択中の機体スキン ID

シングルファイル構成（`index.html` に CSS / JS 全部入り）で、依存は CDN の Three.js のみ。

## ⚙️ 開発・実行

ローカルで動かす場合は静的サーバを立てるだけ：

```bash
# Python
python3 -m http.server 8080

# Node.js
npx serve .
```

ブラウザで `http://localhost:8080` を開く。

> ⚠️ Service Worker は `file://` プロトコルでは動きません。動作確認は必ず HTTP 経由で。

### ファイル構成

```
hover-dodge/
├── index.html       ← ゲーム本体（HTML + CSS + JS 全部入り）
├── sw.js            ← Service Worker（オフラインキャッシュ）
├── manifest.json    ← PWA マニフェスト
├── icon.svg         ← ベクターアイコン（manifest の primary）
├── icon-180.png     ← iOS apple-touch-icon (180x180)
├── icon-192.png     ← Android maskable (192x192)
├── icon-512.png     ← Android maskable (512x512)
├── make-icons.html  ← アイコン PNG 再生成用ページ
├── README.md
└── ...
```

### バージョン管理

リリースごとに以下を同時に更新します：

- `index.html` 内 `<p class="rot-version">SYS_REV  X.Y.Z</p>`
- `sw.js` 内 `const CACHE = 'hoverdodge-vX.Y.Z';`

SW キャッシュキーを上げることで、既存 PWA ユーザーに新版が確実に配信されます。

アイコンを変更した場合は `make-icons.html` をブラウザで開いて PNG を再生成・差し替えてください（iOS の `apple-touch-icon` は SVG を受け付けないため PNG 必須）。

## 🎨 ビジュアル特徴

- **ネオン・グロー** — UnrealBloomPass による HDR 発光
- **シップ** — Bezier 曲線で構築された船体・翼・コクピット、被弾判定はスキルで変化
- **壁** — マゼンタ系ネオン + パルスエッジ
- **敵 2 種** — `BULLET`（リング付きプラズマ球）/ `SEEKER`（旋回速度キャップ付き追跡矢印型）
- **PULSE** — 二重拡張リング + バーストスパーク
- **SHIELD ブレイク** — 緑のリングで周囲の敵を巻き込む防衛発火
- **前方弾** — シアンの小さな弾、衝突で敵をスパーク化
- **背景** — スターフィールド + ニュアンスネビュラ + 動的グリッド + スピードライン
- **縦画面待機** — ブランドロゴ + タクティカルスコープ + フェーズ進行 + 自己ベスト + ショップ

## 📱 対応環境

- iOS Safari 15.4+ / Chrome / Edge / Firefox
- 横画面（landscape）専用 — 縦持ち時はオーバーレイで誘導
- Safe Area（ノッチ・ホームインジケーター）対応
- 60 / 120 / 144 Hz 端末で同一の難易度（dt ベースのタイムステップ）
