HOW TO PLAY

▸ 基本操作

タップ / SPACE
自機を上昇させる（重力で落下）
Z / PULSE
範囲攻撃。CDあり (5秒)
M
音 (BGM/SE) ON / OFF
ESC
GAME OVER 画面からタイトルへ

▸ 目的

壁と敵をよけて長く生存する。スコア=生存秒数+ボーナス。
死んだら RETRY で即再挑戦。自己ベストはローカルに保存。

▸ スコア・ボーナス

+10
壁スレスレ通過（隙間の外側30%以内）
+25
極スレスレ NEAR! （外側15%以内）
+15
PULSE で弾型 (BULLET) 撃破
+20
PULSE で追跡型 (SEEKER) 撃破

▸ フェーズ進行

生存時間に応じて4段階に難易度UP。BGMもフェーズに合わせて重厚に。

  • PHASE 1 WALLS (0–20s) — 壁のみ
  • PHASE 2 BULLETS (20–50s) — 弾型登場
  • PHASE 3 SEEKERS (50–100s) — 追跡型登場
  • PHASE 4 CHAOS (100s+) — 最高速度

▸ XP & スキルツリー (SHOP)

プレイ中に XP 獲得 → 縦画面の SHOP でスキルを解放。
ルートスキル (FORWARD SHOT / SHIELD / MAGNET) を購入すると
上位スキルが解放され、より強力な効果を装備可能に。
装備枠は最大 2 つ。

▸ SKIN

縦画面の SKIN ボタンから機体カラーを6色変更可能。
CYAN / MAGENTA / EMERALD / FIRE / VOID / GOLD。

▸ コツ

  • 壁の隙間ギリギリを狙うと NEAR ボーナスでスコアと XP が伸びる
  • PULSE は敵が集中した時に温存して使うと効果大
  • PHASE 3 以降は SEEKER が後ろから追跡。急上下で振り切れる
  • SHIELD / PHOENIX を装備すれば最初の被弾を生き残れる
